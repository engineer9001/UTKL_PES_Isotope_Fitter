The purpose of this code is to allow the fast and easily-changable fitting of different combinations of positron-emitting isotopes to PET event rate data.

All files are required to be in the same directory, and to run, you simply run python my_FLASH_Fitting.py although you must supply a valid data file which will not be uploaded here for the sake of security

See the command line arguments file to get an understanding of what can be specified or simply run my_FLASH_fitting --help to have them listed out for you.
